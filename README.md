# 42-Born 2 be Root

***** creation Virtual Machine *****
download Debien from Debien.org

Open VM
bar de menu -> Machine -> New
Name - nrobinso42
choose folder sgoinfre
choose linux
choose Debien64 (downloaded to sgoinfre)
memory 1024mb
Create Virtual harddisk now
choose VDI
choose Dynamically allocated
8 gb disksize
start
install
